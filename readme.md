### 如何部屬

+ 本專案使用之 jdk 以及套件版本請參考 `Maven` 的 `pom.xml` 文件
+ 請安裝 jdk 以及 Maven
+ 在 `src/main/java/com/ruthless/app/property` 資料夾下新增 `telegramConfig.properties` 文件
+ 文件內容應當長這樣：
    ```properties
    telegarm.token=5372662822:[你的 Telegram Token 沒有中括號]
    ```
+ 大功告成！
